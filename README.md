# YukiLLM

本项目主体是由 Yuki 大模型生成的《魔法使之夜》续写可玩 Demo，打开后可以像阅读游戏剧情一样逐句推进文本。


## 启动方法

1. 打开本文件所在的文件夹。
2. 双击 `一键启动.bat`。
3. 等待浏览器自动打开页面。
4. 如果页面显示剧情文字，就说明启动成功。

启动后会出现一个黑色窗口，这是本地服务窗口。Demo 运行时请不要关闭它。




## 其他写作示例

除了这个可玩的《魔法使之夜》续写 Demo，本项目还包含 Yuki 大模型生成的其他写作示例。

这些示例放在 `Yuki大模型生成的其他写作示例` 文件夹中，目前包括：

- `妹居版RE0.txt`
- `妹居版白色相簿2.txt`
- `续写为美好的世界献上祝福！.txt`

直接双击这些 `.txt` 文件即可阅读。
