# YukiLLM

本项目主体是由 Yuki 大模型生成的《魔法使之夜》续写可玩 Demo，及Yuki大模型生成的其他写作示例


## 启动方法

1. 点击页面右上方绿色的 `Code` 按钮。
2. 在弹出的菜单里点击 `Download ZIP`。
3. 等待压缩包下载完成。
4. 找到下载好的 ZIP 压缩包，右键选择“全部解压”或“解压到当前文件夹”。
5. 打开解压后的项目文件夹。
6. 双击 `一键启动.bat`。
7. 等待浏览器自动打开页面。

启动后会出现一个黑色窗口，这是本地服务窗口。Demo 运行时请不要关闭它。


## 其他写作示例

除了这个可玩的《魔法使之夜》续写 Demo，本项目还包含 Yuki 大模型生成的其他写作示例。

这些示例放在 `Yuki大模型生成的其他写作示例` 文件夹中，目前包括：

- `妹居版RE0.txt`
- `妹居版白色相簿2.txt`
- `续写为美好的世界献上祝福！.txt`

直接双击这些 `.txt` 文件即可阅读。
