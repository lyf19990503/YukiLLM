# AiTypeMoon Demo 启动说明

这是一个本地网页 Demo。你不需要会写代码，也不需要安装开发工具，按下面步骤操作即可启动。

## 一键启动

1. 打开本文件所在的文件夹。
2. 双击 `一键启动.bat`。
3. 等待浏览器自动打开页面。
4. 如果页面显示剧情文字，就说明启动成功。

启动后会出现一个黑色窗口，这是本地服务窗口。Demo 运行时请不要关闭它。

## 如何操作 Demo

- 鼠标左键点击画面：继续下一句。
- 键盘空格或回车：继续下一句。
- 鼠标右键：切换自动播放。
- 鼠标滚轮：快进或回退。

## 如何关闭

关闭浏览器页面后，再回到黑色窗口：

- 按 `Ctrl + C` 停止服务。
- 或者直接关闭黑色窗口。

## 常见问题

### 双击 `index.html` 可以吗？

不建议。这个 Demo 需要读取 `剧本.txt`，直接双击 `index.html` 可能会被浏览器拦截，导致显示“剧本加载失败”。请使用 `一键启动.bat`。

### 双击后浏览器没有自动打开

看黑色窗口里显示的地址，例如：

```text
http://127.0.0.1:5173/index.html
```

把这个地址复制到浏览器地址栏打开即可。

### 端口不是 5173 正常吗？

正常。如果 `5173` 已经被其他程序占用，启动脚本会自动换成 `5174`、`5175` 等可用端口。

### Windows 提示不安全或阻止运行

这是因为脚本来自本地文件夹，Windows 有时会提醒。确认文件来自本项目后，可以选择“仍要运行”。

### 页面一直显示“剧本加载失败”

请确认：

- 是通过 `一键启动.bat` 启动的。
- 黑色窗口没有被关闭。
- `剧本.txt` 还在项目文件夹里。

## 文件说明

- `一键启动.bat`：双击启动 Demo。
- `scripts/start.ps1`：本地服务脚本。
- `index.html`：网页入口。
- `game.js`：Demo 逻辑。
- `styles.css`：页面样式。
- `剧本.txt`：剧情文本。

本项目为技术演示，剧本内容由 Yuki 大模型生成。
